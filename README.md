# fecss-blog-vue

> 一个基于vue.js,koa和mongodb的个人博客系统。

## 简介

``fecss-blog``是使用nodejs+mongodb+vue的个人博客系统。

**此为客户端, 需要配合[服务端](https://github.com/wendaosanshou/fecss-blog-api)使用**

> 需要预先安装并运行服务端，再运行客户端

## 运行步骤

``` bash
git clone git@github.com:wendaosanshou/fecss-blog.git

cd fecss-blog

npm install

npm run dev
```
