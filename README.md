# fecss-blog-vue

> 一个基于vue.js,koa和mongodb的个人博客系统。

**此为客户端, 需要配合服务端使用**

服务端地址：[fecss-blog-api](https://github.com/wendaosanshou/fecss-blog-api)
> 需要预先安装并运行服务端，再运行客户端

## 运行步骤

``` bash
git clone git@github.com:wendaosanshou/fecss-blog.git

cd fecss-blog-vue

npm install

npm run dev
```
